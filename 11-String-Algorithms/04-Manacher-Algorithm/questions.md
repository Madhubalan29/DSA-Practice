# 🔥 Interview & CP Questions — Manacher Algorithm

## Question 1: Longest Palindromic Substring
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/longest-palindromic-substring/
- **Difficulty:** Medium
- **Description:** Find the longest palindromic substring.
- **Key Concepts:** Manacher's, Expand Around Center
- **Why Interview-Worthy:** Manacher's gives O(n) — but expand-around-center is also accepted.

## Question 2: Palindromic Substrings
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/palindromic-substrings/
- **Difficulty:** Medium
- **Description:** Count the number of palindromic substrings.
- **Key Concepts:** Manacher's
- **Why Interview-Worthy:** Counting version — Manacher's gives all palindrome radii.

## Question 3: Longest Palindromic Subsequence

- **Description:** Find the longest palindromic subsequence.
