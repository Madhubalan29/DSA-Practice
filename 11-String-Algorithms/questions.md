# 🔥 Interview Hard Questions — String Algorithms

## Question 1: Minimum Window Substring (Repeated — see also 06)
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/longest-valid-parentheses/
- **Difficulty:** Hard
- **Problem:** Longest Valid Parentheses — Given a string containing only '(' and ')', return the length of the longest valid (well-formed) parentheses substring.
- **Key Concepts:** Stack-based approach, DP, String parsing
- **Why Interview-Worthy:** Classic string + stack hard problem. Tests multiple solution approaches.

## Question 2: Distinct Subsequences
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/distinct-subsequences/
- **Difficulty:** Hard
- **Description:** Given two strings s and t, return the number of distinct subsequences of s which equals t.
- **Key Concepts:** DP on Strings, 2D DP, Subsequence counting
- **Why Interview-Worthy:** Fundamental string DP problem — tests understanding of subsequence state transitions.
