# 🔥 Interview & CP Questions — Rabin Karp

## Question 1: Repeated DNA Sequences
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/repeated-dna-sequences/
- **Difficulty:** Medium
- **Description:** Find all 10-letter sequences that occur more than once.
- **Key Concepts:** Rolling Hash, Rabin-Karp
- **Why Interview-Worthy:** Rolling hash for fixed-length substring matching.
