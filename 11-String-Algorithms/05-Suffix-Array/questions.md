# 🔥 Interview & CP Questions — Suffix Array

## Question 1: Suffix Array Construction
- **Platform:** CP-Algorithms
- **Link:** https://cp-algorithms.com/string/suffix-array.html
- **Difficulty:** Hard
- **Description:** Build suffix array in O(n log n) or O(n log² n).
- **Key Concepts:** Suffix Array
- **Why Interview-Worthy:** Foundation for many string problems.

---

## Question 2: Distinct Substrings
- **Platform:** SPOJ
- **Link:** https://www.spoj.com/problems/DISUBSTR/
- **Difficulty:** Medium
- **Description:** Count distinct substrings using suffix array + LCP.
- **Key Concepts:** Suffix Array, LCP
- **Why Interview-Worthy:** n*(n+1)/2 - sum(LCP) gives distinct substring count.

---

## Question 3: Longest Common Substring
- **Platform:** SPOJ
- **Link:** https://www.spoj.com/problems/LCS/
- **Difficulty:** Hard
- **Description:** Find the longest common substring of two strings.
- **Key Concepts:** Suffix Array, LCP
- **Why Interview-Worthy:** Concatenate strings and use SA + LCP.

---

## Question 4: String Period
- **Platform:** CSES
- **Link:** https://cses.fi/problemset/task/1733
- **Difficulty:** Medium
- **Description:** Find all periods of a string.
- **Key Concepts:** Suffix Array, Z-Function
- **Why Interview-Worthy:** Period = n - Z[i] or n - failure[n].
