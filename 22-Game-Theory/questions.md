# 🔥 Interview Hard Questions — Game Theory

## Question 1: Stone Game III
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/stone-game-iii/
- **Difficulty:** Hard
- **Description:** Alice and Bob take turns picking 1, 2, or 3 stones from the front. Each player plays optimally. Return "Alice", "Bob", or "Tie" based on who wins.
- **Key Concepts:** Minimax, DP (suffix DP), Game theory optimal play
- **Why Interview-Worthy:** Classic game theory + DP problem. Tests minimax thinking.

## Question 2: Cat and Mouse
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/cat-and-mouse/
- **Difficulty:** Hard
- **Description:** A game on an undirected graph. Mouse starts at node 1, Cat at node 2. Mouse wins by reaching node 0, Cat wins by catching Mouse. Determine the winner with optimal play.
- **Key Concepts:** Game Theory, BFS from terminal states, State-space search, Minimax
- **Why Interview-Worthy:** Advanced game theory on graphs — one of the hardest problems on LeetCode.
