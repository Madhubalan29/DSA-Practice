# 🔥 Interview Hard Questions — Fenwick Tree (BIT)

## Question 1: Count of Smaller Numbers After Self (BIT approach)
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/count-of-smaller-numbers-after-self/
- **Difficulty:** Hard
- **Description:** Given an integer array nums, return a new counts array where counts[i] is the number of smaller elements to the right of nums[i]. Solve using BIT.
- **Key Concepts:** Fenwick Tree / BIT, Coordinate compression, Reverse traversal
- **Why Interview-Worthy:** Classic BIT application — alternative to merge sort approach.

## Question 2: Create Sorted Array through Instructions
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/create-sorted-array-through-instructions/
- **Difficulty:** Hard
- **Description:** Given an integer array instructions, create a sorted array by inserting elements one by one. Cost of each insertion = min(elements strictly less than, elements strictly greater than current). Return total cost mod 10^9+7.
- **Key Concepts:** BIT for order statistics, Online insertion cost
- **Why Interview-Worthy:** Advanced BIT usage for dynamic order statistics.
