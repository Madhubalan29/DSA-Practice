# 🔥 Interview & CP Questions — Basic Binary Search

## Question 1: Binary Search
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/binary-search/
- **Difficulty:** Easy
- **Description:** Given a sorted array and target, search for target in nums.
- **Key Concepts:** Binary Search
- **Why Interview-Worthy:** Fundamental algorithm, basis for all binary search variations.

---

## Question 2: Search Insert Position
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/search-insert-position/
- **Difficulty:** Easy
- **Description:** Given a sorted array, return the index if target is found or where it would be inserted.
- **Key Concepts:** Binary Search, Lower Bound
- **Why Interview-Worthy:** Tests understanding of lower bound and insert positions.

## Question 3: First Bad Version
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/first-bad-version/

## Question 4: Find First and Last Position
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/find-first-and-last-position-of-element-in-sorted-array/
