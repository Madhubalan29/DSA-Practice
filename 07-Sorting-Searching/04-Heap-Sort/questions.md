# 🔥 Interview & CP Questions — Heap Sort

## Question 1: Sort an Array
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/sort-an-array/
- **Difficulty:** Medium
- **Description:** Implement heap sort from scratch.
- **Key Concepts:** Heap Sort
- **Why Interview-Worthy:** In-place O(n log n) sort — good for understanding heaps.

## Question 2: Kth Largest Element
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/kth-largest-element-in-an-array/
- **Difficulty:** Medium
- **Description:** Find kth largest using a min-heap of size k.
- **Key Concepts:** Heap
- **Why Interview-Worthy:** Min-heap of size k — O(n log k) approach.
