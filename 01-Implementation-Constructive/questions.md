# 🔥 Interview Hard Questions — Implementation & Constructive

## Question 1: Text Justification
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/text-justification/
- **Difficulty:** Hard
- **Description:** Given an array of words and a maxWidth, format the text such that each line has exactly maxWidth characters and is fully (left and right) justified. Greedy packing with precise whitespace distribution.
- **Key Concepts:** Simulation, Greedy, String Manipulation
- **Why Interview-Worthy:** Tests edge-case handling, string formatting precision, and clean code under complex constraints.

## Question 2: First Missing Positive
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/first-missing-positive/
- **Difficulty:** Hard
- **Description:** Given an unsorted integer array nums, return the smallest missing positive integer. Must run in O(n) time and O(1) auxiliary space.
- **Key Concepts:** In-place Hashing, Index Mapping, Constructive
- **Why Interview-Worthy:** Classic hard problem testing in-place manipulation and constructive algorithm design.
