# 🔥 Interview & CP Questions — DSU Problems

## Question 1: Making a Large Island
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/making-a-large-island/
- **Difficulty:** Hard
- **Description:** Flip at most one 0 to 1 to get the largest island.
- **Key Concepts:** DSU, Grid
- **Why Interview-Worthy:** DSU with component size tracking.

## Question 2: Most Stones Removed with Same Row or Column
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/most-stones-removed-with-same-row-or-column/
- **Difficulty:** Medium
- **Description:** Remove stones that share a row or column.
- **Key Concepts:** DSU
- **Why Interview-Worthy:** Count components — removed = total - components.

## Question 3: Swim in Rising Water
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/swim-in-rising-water/
- **Difficulty:** Hard
- **Description:** Find min time to swim from top-left to bottom-right.
- **Key Concepts:** DSU, Binary Search
- **Why Interview-Worthy:** Sort cells by elevation, union adjacent, check connectivity.
