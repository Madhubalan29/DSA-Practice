# 🔥 Interview & CP Questions — Catalan Numbers

## Question 1: Unique Binary Search Trees
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/unique-binary-search-trees/
- **Difficulty:** Medium
- **Description:** Given n, how many structurally unique BSTs can store values 1..n?
- **Key Concepts:** Catalan Numbers, DP
- **Why Interview-Worthy:** Direct application of Catalan number formula.

- **Link:** https://leetcode.com/problems/generate-parentheses/
