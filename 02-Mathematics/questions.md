# 🔥 Interview Hard Questions — Mathematics

## Question 1: Count Primes (Optimized Sieve + Number Theory)
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/max-points-on-a-line/
- **Difficulty:** Hard
- **Problem:** Max Points on a Line — Given n points on a 2D plane, find the maximum number of points that lie on the same straight line.
- **Key Concepts:** GCD for slope representation, Hashing, Edge cases (vertical/duplicate)
- **Why Interview-Worthy:** Combines number theory (GCD) with geometry and careful edge-case handling.

## Question 2: Number of Digit One
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/number-of-digit-one/
- **Difficulty:** Hard
- **Description:** Given an integer n, count the total number of digit 1 appearing in all non-negative integers ≤ n.
- **Key Concepts:** Mathematical pattern, Digit counting, Positional analysis
- **Why Interview-Worthy:** Requires deep mathematical insight and pattern recognition across digit positions.
