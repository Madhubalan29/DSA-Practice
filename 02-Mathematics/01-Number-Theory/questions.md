- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/ugly-number-ii/
- **Difficulty:** Medium
- **Description:** Find the n-th ugly number (prime factors only include 2, 3, 5).

## Question 2: Happy Number
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/happy-number/
- **Difficulty:** Easy
- **Description:** Determine if a number is a happy number (sum of squares of digits eventually reaches 1).

## Question 3: Perfect Squares
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/perfect-squares/
- **Difficulty:** Medium
- **Description:** Given an integer n, return the least number of perfect square numbers that sum to n.
- **Key Concepts:** Number Theory, BFS, DP
- **Why Interview-Worthy:** Lagrange's four-square theorem — math insight helps.
