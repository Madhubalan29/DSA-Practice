# Greedy Algorithms

## Core Idea
Make locally optimal choice at each step hoping for global optimum.

## Key Patterns
- **Activity Selection**: Sort by end time, greedily pick non-overlapping
- **Interval Scheduling**: Maximize non-overlapping intervals
- **Huffman Coding**: Build optimal prefix-free encoding tree

## When Greedy Works
- Problem has **optimal substructure** AND **greedy choice property**
- Prove greedy is correct using exchange argument

## Big Company Problems
- Jump game I & II
- Gas station
- Candy distribution
- Task scheduler
- Non-overlapping intervals
- Minimum platforms
