# 🔥 Interview & CP Questions — Network Flow

## Question 1: Maximum Flow (Ford-Fulkerson)
- **Platform:** CP-Algorithms
- **Link:** https://cp-algorithms.com/graph/edmonds_karp.html
- **Difficulty:** Hard
- **Description:** Find maximum flow using Edmonds-Karp (BFS-based Ford-Fulkerson).
- **Key Concepts:** Network Flow
- **Why Interview-Worthy:** Foundation of flow algorithms.

## Question 3: Maximum Bipartite Matching
- **Platform:** CP-Algorithms
- **Link:** https://cp-algorithms.com/graph/kuhn_algorithm.html
- **Difficulty:** Hard
- **Description:** Find maximum matching in a bipartite graph.
- **Key Concepts:** Hungarian/Kuhn's Algorithm
- **Why Interview-Worthy:** Reduce to max flow or use augmenting paths.

## Question 4: Min Cut
- **Platform:** CP-Algorithms
- **Link:** https://cp-algorithms.com/graph/edmonds_karp.html
- **Difficulty:** Hard
- **Description:** Find minimum cut (equals max flow by max-flow min-cut theorem).
