# 🔥 Interview Hard Questions — Graph Algorithms

## Question 1: Word Ladder II
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/word-ladder-ii/
- **Difficulty:** Hard
- **Description:** Given beginWord, endWord, and a wordList, find all shortest transformation sequences from beginWord to endWord, such that only one letter can be changed at a time and each transformed word must exist in the wordList.
- **Key Concepts:** BFS for shortest path, DFS/Backtracking for all paths, Graph construction
- **Why Interview-Worthy:** Combines BFS + DFS — one of the hardest graph problems on LeetCode.

## Question 2: Critical Connections in a Network
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/critical-connections-in-a-network/
- **Difficulty:** Hard
- **Description:** Given n servers and connections between them, find all critical connections (bridges) — edges whose removal disconnects the graph.
- **Key Concepts:** Tarjan's Bridge-Finding Algorithm, DFS, Low-Link values
- **Why Interview-Worthy:** Tests knowledge of advanced graph algorithms. Asked at Amazon, Google.
