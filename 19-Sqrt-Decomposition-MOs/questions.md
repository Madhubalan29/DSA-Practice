# 🔥 Interview Hard Questions — Sqrt Decomposition / MO's Algorithm

## Question 1: Range Sum Query — MO's Algorithm (Offline)
- **Platform:** Codeforces
- **Link:** https://codeforces.com/problemset/problem/86/D (Powerful Array)
- **Difficulty:** Hard
- **Description:** Given an array and Q queries [L, R], for each query compute Σ(cnt[x]^2 * x) where cnt[x] is the count of value x in [L, R]. Process offline using MO's algorithm.
- **Key Concepts:** MO's Algorithm, Sqrt Decomposition, Block sorting
- **Why Interview-Worthy:** Fundamental MO's problem — tests offline query optimization skills.

## Question 2: Range Minimum Query (Sqrt Decomposition)
- **Platform:** Codeforces
- **Link:** https://codeforces.com/edu/course/2/lesson/9/3/practice/contest/307094/problem/A
- **Alternative:** SPOJ RMQSQ - https://www.spoj.com/problems/RMQSQ/
- **Difficulty:** Hard
- **Description:** Answer multiple range minimum queries on a static array using sqrt decomposition or sparse table.
- **Key Concepts:** Sqrt Decomposition, Block precomputation, O(1) query with O(sqrt(n)) update
- **Why Interview-Worthy:** Foundation for understanding block-based data structures.
