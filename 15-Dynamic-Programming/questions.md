# 🔥 Interview Hard Questions — Dynamic Programming

## Question 1: Edit Distance
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/edit-distance/
- **Difficulty:** Hard
- **Description:** Given two strings word1 and word2, return the minimum number of operations (insert, delete, replace) required to convert word1 to word2.
- **Key Concepts:** 2D DP, String DP, Classical edit distance (Levenshtein)
- **Why Interview-Worthy:** Foundational DP problem — basis for diff algorithms, spell checkers, DNA sequencing.

## Question 2: Burst Balloons
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/burst-balloons/
- **Difficulty:** Hard
- **Description:** Given n balloons with numbers, burst them to collect coins. Bursting balloon i gives nums[i-1]*nums[i]*nums[i+1] coins. Find the maximum coins.
- **Key Concepts:** Interval DP, Reverse thinking (last balloon burst), Matrix Chain style
- **Why Interview-Worthy:** Classic interval DP — tests ability to reframe the problem (think about which balloon is burst LAST).
