# 🔥 Interview Hard Questions — Linked List

## Question 1: Merge k Sorted Lists
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/merge-k-sorted-lists/
- **Difficulty:** Hard
- **Description:** Merge k sorted linked lists and return it as one sorted list.
- **Key Concepts:** Min-Heap/Priority Queue, Divide and Conquer, Linked List manipulation
- **Why Interview-Worthy:** Classic hard problem combining heap + linked list — asked at Google, Amazon, Meta.

## Question 2: Reverse Nodes in k-Group
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/reverse-nodes-in-k-group/
- **Difficulty:** Hard
- **Description:** Given the head of a linked list, reverse the nodes of the list k at a time and return the modified list. Nodes not part of a complete group remain as-is.
- **Key Concepts:** Linked List reversal, Group processing, Pointer manipulation
- **Why Interview-Worthy:** Tests mastery of pointer manipulation under complex constraints.
