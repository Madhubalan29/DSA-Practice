# 🔥 Interview Hard Questions — Bit Manipulation

## Question 1: Maximum XOR of Two Numbers in an Array
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/maximum-xor-of-two-numbers-in-an-array/
- **Difficulty:** Hard (Medium on LC, but the Trie-based approach is interview-hard)
- **Description:** Given an integer array nums, return the maximum XOR of two numbers in the array. Must solve in O(n) time.
- **Key Concepts:** Bitwise Trie, Greedy bit selection, XOR properties
- **Why Interview-Worthy:** Classic combination of bit manipulation + trie data structure.

## Question 2: Minimum Number of Operations to Make Array Continuous
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/minimum-number-of-operations-to-make-array-continuous/
- **Difficulty:** Hard
- **Description:** Given an integer array nums, in one operation you can replace any element with any integer. Return the minimum number of operations to make nums continuous (contains all elements in [x, x+n-1] for some x).
- **Key Concepts:** Sliding Window, Sorting, Bit manipulation concepts for optimization
- **Why Interview-Worthy:** Requires creative transformation of the problem into a sliding window approach.
