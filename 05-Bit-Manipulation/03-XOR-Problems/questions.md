# 🔥 Interview & CP Questions — XOR Problems

## Question 1: Single Number
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/single-number/
- **Difficulty:** Easy
- **Description:** Find the single element where every other appears twice.
- **Key Concepts:** XOR
- **Why Interview-Worthy:** Classic XOR: a ^ a = 0.

- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/single-number-ii/

- **Description:** Every element appears three times except one. Find that one.
