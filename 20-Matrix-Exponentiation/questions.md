# 🔥 Interview Hard Questions — Matrix Exponentiation

## Question 1: Fibonacci Number (Matrix Exponentiation for O(log n))
- **Platform:** LeetCode / Codeforces
- **Link:** https://leetcode.com/problems/fibonacci-number/ (solve in O(log n) via matrix expo)
- **Alternative Hard:** https://codeforces.com/problemset/problem/185/A (Plant)
- **Difficulty:** Hard (due to technique)
- **Description:** Compute the n-th Fibonacci number in O(log n) using matrix exponentiation. |F(n+1) F(n)| = |1 1|^n
- **Key Concepts:** Matrix Exponentiation, Fast power, Linear recurrence
- **Why Interview-Worthy:** Foundation technique for solving any linear recurrence in O(k^3 log n).

## Question 2: Knight Dialer
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/knight-dialer/
- **Difficulty:** Hard (Medium on LC but Hard with matrix expo optimization)
- **Description:** A chess knight on a phone dial pad can press n digits. Count distinct numbers of length n. Optimized with matrix exponentiation.
- **Key Concepts:** Matrix Exponentiation, State transition matrix, Modular arithmetic
- **Why Interview-Worthy:** Practical application of matrix exponentiation to a real-world DP problem.
