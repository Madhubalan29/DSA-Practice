# 🔥 Interview & CP Questions — Graph Powers

## Question 1: Graph Paths I
- **Platform:** CSES
- **Link:** https://cses.fi/problemset/task/1723
- **Difficulty:** Hard
- **Description:** Count paths of exactly k edges using matrix exponentiation.
- **Key Concepts:** Matrix Exponentiation, Graph
- **Why Interview-Worthy:** Adjacency matrix^k gives path counts.
