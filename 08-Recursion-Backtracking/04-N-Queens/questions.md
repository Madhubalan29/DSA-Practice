# 🔥 Interview & CP Questions — N Queens

## Question 1: N-Queens
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/n-queens/
- **Difficulty:** Hard
- **Description:** Place n queens on an n×n chessboard so no two queens attack each other.
- **Key Concepts:** Backtracking
- **Why Interview-Worthy:** THE backtracking problem — every CS student must solve.
