# 🔥 Interview Hard Questions — Recursion & Backtracking

## Question 1: N-Queens II
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/n-queens-ii/
- **Difficulty:** Hard
- **Description:** Return the number of distinct solutions to the n-queens puzzle. Place n queens on an n×n chessboard so that no two queens attack each other.
- **Key Concepts:** Backtracking, Bitmask optimization, Constraint propagation
- **Why Interview-Worthy:** The quintessential backtracking problem — tests pruning, state management, and optimization.

## Question 2: Sudoku Solver
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/sudoku-solver/
- **Difficulty:** Hard
- **Description:** Write a program to solve a Sudoku puzzle by filling the empty cells. Each row, column, and 3×3 sub-box must contain digits 1-9 with no repetition.
- **Key Concepts:** Backtracking, Constraint Propagation, Bitmask for valid states
- **Why Interview-Worthy:** Real-world constraint satisfaction problem — tests systematic search and pruning.
