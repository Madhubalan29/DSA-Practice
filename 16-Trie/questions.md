# 🔥 Interview Hard Questions — Trie

## Question 1: Word Search II
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/word-search-ii/
- **Difficulty:** Hard
- **Description:** Given an m×n board of characters and a list of strings words, return all words on the board. Each word must be formed by sequentially adjacent cells (horizontally or vertically).
- **Key Concepts:** Trie + Backtracking/DFS, Pruning, Board traversal
- **Why Interview-Worthy:** Classic hard Trie problem — combines trie construction with grid DFS.

## Question 2: Palindrome Pairs
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/palindrome-pairs/
- **Difficulty:** Hard
- **Description:** Given a list of unique words, return all pairs of distinct indices (i, j) such that the concatenation words[i] + words[j] is a palindrome.
- **Key Concepts:** Trie with reverse insertion, Palindrome checking, Edge cases
- **Why Interview-Worthy:** Complex trie application — tests deep understanding of trie traversal + palindrome properties.
