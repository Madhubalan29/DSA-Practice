# 🔥 Interview Hard Questions — Geometry

## Question 1: Erect the Fence (Convex Hull)
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/erect-the-fence/
- **Difficulty:** Hard
- **Description:** Given a set of points, return the points that form the convex hull (outermost fence enclosing all trees). Return points in any order.
- **Key Concepts:** Convex Hull (Andrew's Monotone Chain / Graham Scan), Cross product
- **Why Interview-Worthy:** Classic computational geometry problem — tests cross-product and hull construction.

## Question 2: Rectangle Area II
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/rectangle-area-ii/
- **Difficulty:** Hard
- **Description:** Given a list of axis-aligned rectangles, find the total area covered by all rectangles (union of areas, not double-counting overlaps).
- **Key Concepts:** Coordinate Compression, Sweep Line, Segment Tree (optional)
- **Why Interview-Worthy:** Combines sweep line with coordinate compression — a powerful technique for geometry problems.
