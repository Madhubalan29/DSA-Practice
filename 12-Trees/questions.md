# 🔥 Interview Hard Questions — Trees

## Question 1: Binary Tree Maximum Path Sum
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/binary-tree-maximum-path-sum/
- **Difficulty:** Hard
- **Description:** Given the root of a binary tree, return the maximum path sum of any non-empty path. A path can start and end at any node.
- **Key Concepts:** Tree DP, Post-order traversal, Path vs. contribution logic
- **Why Interview-Worthy:** Classic tree hard problem — tests understanding of recursive return value vs. global update.

## Question 2: Serialize and Deserialize Binary Tree
- **Platform:** LeetCode
- **Link:** https://leetcode.com/problems/serialize-and-deserialize-binary-tree/
- **Difficulty:** Hard
- **Description:** Design an algorithm to serialize and deserialize a binary tree. Convert tree to string and reconstruct from string.
- **Key Concepts:** Preorder traversal, Queue-based deserialization, Null markers
- **Why Interview-Worthy:** Tests design thinking + tree traversal — frequently asked at Amazon, Google.
