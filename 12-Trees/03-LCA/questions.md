## Question 5: Distance Queries
- **Platform:** CSES
- **Link:** https://cses.fi/problemset/task/1135
- **Difficulty:** Hard
- **Description:** Find distance between two nodes using LCA.
- **Key Concepts:** LCA, Tree
- **Why Interview-Worthy:** dist(u,v) = depth[u] + depth[v] - 2*depth[lca(u,v)].
